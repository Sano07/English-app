English Learning App

Android приложение для изучения английских слов по уровням и категориям.

Stack: Kotlin, Android SDK, MVVM, Intents  

Архитектура: 
UI → Activity/ViewModel → Data  

- Выбор уровня сложности (Beginner / Intermediate / Advanced)  
- Работа с тематическими словарями  
- Передача данных между экранами  
- Отображение списков слов
